Introduction

The Cloud Whatsapp API allows you to programmatically send messages via authenticated WhatsApp sessions. All endpoints require an API Key specifically generated for your device.

Authentication

Pass your API Key in the Authorization header as a Bearer token.

Authorization: Bearer YOUR_API_KEY

Sending Text Messages

Send plain text messages to any WhatsApp number.

POST /api/messages/send
Parameter Type Description
deviceId string The unique ID of the session
number string Recipient phone number (with country code)
message string The text message content
